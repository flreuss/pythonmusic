Synth
=====

The ``synth`` module contains the :obj:`Synth <pythonmusic.synth.Synth>` and :obj:`SynthPlayer <pythonmusic.synth.SynthPlayer>` objects that 
are used to playback scores on your device.

.. important:: This module is only available if `fluidsynth` has been found on your system. Refer to the
   :doc:`installation instructions <../intro/installation>` for more information.
