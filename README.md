# PythonMusic
*make music with computers*

## Requirements
- Python >=3.7

Other dependencies should be installed automatically:
- mido
- pygame
- python-osc

## Install
For now, this package needs to be installed manually. Use the following command to add `pythonmusic` to your virtual environment:

```bash
# source venv/bin/activate
pip install -e path/to/pythonmusic
```
